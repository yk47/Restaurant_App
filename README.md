# Restaurant_App

Simple Restaurant App showing list and details of restaurants built using Flutter.

---

## 📱 Project Overview

This is a Flutter-based mobile application that allows users to:

* View a list of restaurants
* See detailed information about each restaurant
* Perform search and filtering (if implemented)

---

## 🚀 Getting Started

### Prerequisites

* Flutter SDK installed
* Android Studio / VS Code
* Emulator or physical device

### Installation

```bash
git clone https://github.com/yk47/Restaurant_App.git
cd Restaurant_App
flutter pub get
```

### Run the app

```bash
flutter run
```
